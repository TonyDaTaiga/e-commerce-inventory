# E-commerce Inventory System

A robust inventory management system built with Python, SQLAlchemy, and Flask.

## 🚀 Features

- User Authentication & Authorization
- Product Management
- Order Processing
- Inventory Tracking
- Reports Generation
- RESTful API
- Admin Dashboard
## 🛠️ Technologies

- **Backend:**
  - Python 3.8+
  - Flask
  - SQLAlchemy
  - Alembic (migrations)
  - JWT Authentication

- **Database:**
  - PostgreSQL
  - Redis (caching)

- **Frontend:**
  - HTML5/CSS3
  - JavaScript
  - Bootstrap 5

## 📋 Prerequisites

- Python 3.8+
- PostgreSQL
- Redis
- Node.js (for frontend builds)
